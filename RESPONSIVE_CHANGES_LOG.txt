HOW TO USE RESPONSIVE UTILITIES
================================================================================

Responsive Text:
  <p class="text-responsive-lg">Large responsive text</p>
  <p class="text-responsive-md">Medium responsive text</p>
  <p class="text-responsive-sm">Small responsive text</p>

Responsive Padding:
  <div class="p-responsive">Content with responsive padding</div>

Responsive Margin:
  <div class="m-responsive">Content with responsive margin</div>

Responsive Gap (Flexbox/Grid):
  <div style="display: flex; gap-responsive">Items here</div>

Writing New Media Queries:
  @media (max-width: 767px) { /* Mobile */ }
  @media (min-width: 768px) and (max-width: 1023px) { /* Tablet */ }
  @media (min-width: 1024px) { /* Desktop */ }
  @media (min-width: 1700px) { /* Large Desktop */ }
